About
---
- Built CP Analyzer
-  a full-stack web application that analyzes Codeforces performance
- Converts raw submission history into meaningful insights (accuracy, rating strength, weak topics)
- Helps competitive programmers identify improvement areas and target next rating milestone
- Designed clean dashboard UI with interactive visual charts
---
Features
---
- Performance statistics (Solved, Accuracy, Avg Rating, Max Rating)
- Topic-wise strength & weakness detection
- Graphical analysis using charts
- Programming language usage breakdown
- Suggested target rating based on performance
- Real-time data fetched from Codeforces API
---

---
 Frontend

Other
---

---
Project Purpose
---
- This project helps competitive programmers stop blind practice and instead practice data-driven problem solving by understanding:
- Which topics they over-practice
- Which topics they avoid
- How far they are from next rating level
